# AI-Driven Development Demo Game

이 프로젝트는 초기 기획부터 구현까지 전 과정에 AI를 적극적으로 활용한 데모 게임입니다.

## 🚀 Project Overview

본 데모 게임은 에셋 리소스 준비와 최종 빌드 작업을 제외한 모든 개발 공정을 AI가 수행했습니다. 개발자는 직접적인 코드 작성 대신 리팩토링, 디버깅 등 모든 기술적 해결 과정을 AI(Claude Code, Codex)를 통해 진행하며 프로젝트를 관리했습니다.
